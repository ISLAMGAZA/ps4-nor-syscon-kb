# قاعدة المجتمع (تقارير مُجهّلة)

ساعد في بناء corpus مفتوح يربط الأعطال بكل موديل/FW. استخدم زر **تصدير التقرير** في الأداة (أو `main.export_report`) لإنتاج ملف JSON لا يحتوي أي سيريال أو MAC أو بيانات قرص - فقط الموديل وFW وحالة الصحة وقائمة المناطق المعطوبة.

أرسله إلى `community/corpus/` ليُجمّع في إحصاءات عامة. انظر `community/SCHEMA.md` للصيغة الدقيقة و`community/CONTRIBUTING.md` لطريقة المساهمة.
